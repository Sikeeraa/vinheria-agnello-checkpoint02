 📖 Sobre o Projeto
 
A **Vinharia Agnello** é uma aplicação desenvolvida para realizar o cadastro e gerenciamento de vinhos de forma simples e interativa.
 
O sistema permite cadastrar diferentes rótulos de vinho, armazenando informações importantes como:
 
- Nome do vinho

- Tipo do vinho

- Safra

- Quantidade em estoque
-Idade
Além disso, o projeto realiza validações automáticas, classificação dos vinhos conforme a idade da safra e geração de relatórios finais.
 
O principal objetivo do projeto é praticar conceitos fundamentais de lógica de programação utilizando JavaScript.

 ⚙️ Como Funciona
 
 🍷 Cadastro de Vinhos
 
Ao clicar no botão de cadastro:
 
1. O sistema inicia o processo de cadastro

2. O usuário informa:

   - Nome do vinho

   - Tipo

   - Safra

   - Quantidade em estoque                                                                                                                                          - Idade do vinho

3. O sistema realiza:

   - Validação dos dados inseridos

   - Verificação de campos vazios

   - Classificação automática do vinho

   - Controle de estoque
 
---
 
 Classificação dos Vinhos
 
Os vinhos são classificados automaticamente com base na safra informada.
 
Exemplo:
 
- Vinho jovem

- Vinho amadurecido

- Vinho antigo
 
---
 
 📦 Controle de Estoque
 
O sistema verifica automaticamente a quantidade disponível em estoque.
 
Caso o estoque esteja baixo, um alerta é exibido ao usuário.
 
---
 
  Relatório Final
 
Ao final dos cadastros, o sistema apresenta um relatório contendo:
 
- Quantidade total de vinhos cadastrados

- Informações dos rótulos

- Status do estoque

- Classificação dos vinhos
